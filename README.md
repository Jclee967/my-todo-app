# my-todo-app
This is a simple todo application that is able to display all todo items and highlight the selected iteam.

### Built With
* [![React][React.js]][React-url]
* [![Bootstrap][Bootstrap.com]][Bootstrap-url]

## Installation
```bash
npm install
```

## Getting started
```bash
npm run dev
```
